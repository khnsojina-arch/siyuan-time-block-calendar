# 时间块日历｜思源笔记日程规划与时间管理插件

**时间块日历** 是一个面向思源笔记的日程规划与时间管理插件，提供类似 Notion Calendar 的时间块日历视图，帮助你在思源笔记中管理学习计划、复习安排、工作任务、待办事项、提醒和番茄钟记录。

无论你是想安排考研复习、规划每日任务、管理项目进度，还是用时间块方法提升专注效率，都可以通过这个插件把「计划」和「笔记」连接起来。

## 一句话介绍

在思源笔记中使用时间块日历完成日程规划、任务计划、提醒、番茄钟和 JSON 备份管理。

## 核心功能

- **多视图日历**：支持周视图、三天视图、月视图和年视图，适合从当天任务到长期复习计划的不同粒度规划。
- **时间块管理**：通过拖拽创建时间块，移动时间块调整日期和开始时间，拖动边缘调整开始与结束时间。
- **任务计划与待办**：为时间块设置标题、完成状态、标签、颜色和备注，把任务安排清晰地放进日历。
- **提醒与通知**：支持时间块提醒和桌面系统通知，适合学习计划、会议、复习节点和日常待办提醒。
- **番茄钟**：侧栏内置轻量番茄钟，便于结合时间管理和专注执行。
- **导入 / 导出**：支持 JSON 备份导入导出，也支持将 iHour `.ihbak` 解析导出为 JSON。
- **思源数据联动**：时间块会保存为思源文档中的记录表格，并同步写入块属性，便于刷新和重新打开后恢复日程。

## 适合谁使用

- 需要在思源笔记中做日程规划和时间管理的用户。
- 希望用时间块方法安排每日任务、学习计划或复习计划的学生和备考用户。
- 需要把工作任务、项目计划、会议安排和待办事项集中管理的职场用户。
- 喜欢 Notion Calendar 风格，但希望计划数据留在思源笔记里的用户。
- 需要提醒、番茄钟和备份导出能力的长期规划用户。

## 使用场景

- **学习计划**：把课程、阅读、刷题、背单词等安排到具体时间段。
- **复习计划**：按周视图或月视图规划考研、考试、证书复习节奏。
- **工作计划**：安排会议、深度工作、项目任务和每日待办。
- **生活管理**：记录运动、用餐、休息、家务和个人事务。
- **专注执行**：结合番茄钟，把计划好的时间块转化为实际专注行动。

## 功能亮点

- 类 Notion Calendar 的时间块体验，同时保留思源笔记的数据组织方式。
- 周视图适合细粒度安排，月视图适合查看整体任务分布，年视图适合回顾长期节奏。
- 支持拖拽创建、移动和调整时间块，适合快速修改日程。
- 标签、颜色、备注和完成状态可组合使用，便于区分学习、工作、生活和复习任务。
- 提醒和桌面系统通知可以帮助你减少遗漏。
- JSON 导入导出方便备份、迁移和恢复。

## 基本使用方法

安装并启用插件后，点击思源顶部栏的时间块日历图标即可打开日历。

桌面端会以标签页方式打开日历；移动端会使用弹窗入口打开日历。你可以在日历空白区域拖拽创建时间块，也可以拖动已有时间块调整日期、开始时间和结束时间。

插件默认将时间块写入配置中的日记根路径，例如：

```text
/daily note/YYYY/MM/YYYY-MM-DD
```

时间块会以记录表格和块属性的形式保存在思源中。刷新、重启或重新打开日历后，插件会从思源文档和块属性中重新读取数据。

## 导入 / 导出说明

- **JSON 导出**：可导出插件设置、标签、时间块和提醒信息，用于备份或迁移。
- **JSON 导入**：可从备份文件恢复时间块数据，导入时会尽量避免重复写入。
- **iHour 导出**：可选择 iHour `.ihbak` 文件，将其解析并导出为 JSON，方便后续统计或交给 AI 分析。

## 数据安全说明

插件通过思源内核 API 读写文档、块属性和插件配置，不直接操作思源工作空间的数据文件。

删除时间块会删除对应的日历记录。执行删除前，插件会显示确认提示。插件卸载时会清理插件自身保存的配置数据。
