# AI 日程规划与 AI 时间块日历

一个面向思源笔记的 **AI 时间块日历插件**：把自然语言、考试安排、学习计划快速解析成时间块、倒数日、正向日和纪念日，适合做日程规划、复习节奏管理、番茄钟专注和提醒。

插件会把时间块写入思源文档中的记录表格，并通过块属性读取和同步；倒数日、纪念日等重要日期保存在插件本地 JSON 中。AI 只负责解析，所有创建结果都需要你预览并确认后才会写入。

## 主要功能

- **AI 日程规划**：输入一句话或粘贴一段安排，AI 自动识别时间块、考试倒数日、正向日和纪念日。
- **AI 时间块创建**：例如“明天上午 9 点到 11 点背单词，分类英语”，识别后预览确认，一键写入日历。
- **AI 倒数日创建**：例如“6 月 13 日六级考试，创建倒数日”，自动识别日期、类型和分类。
- **多视图日历**：周视图、三天视图、月视图、年视图。
- **时间块管理**：拖拽创建、移动、调整时长，支持标题、标签、颜色、备注和完成状态。
- **倒数日**：支持倒数日、正向日、纪念日、已过天数。
- **提醒通知**：支持时间块提醒和桌面系统通知。
- **番茄钟**：侧栏内置轻量番茄钟，可记录专注时间和备注。
- **JSON 导入导出**：支持插件数据备份、迁移和恢复。

## 适合场景

- **学习计划**：背单词、刷题、课程复习、考试安排。
- **考研/考试管理**：倒数日、复习时间块、每日计划、阶段目标。
- **工作计划**：会议、深度工作、项目任务、每日待办。
- **长期规划**：阶段目标、纪念日、持续打卡、重要日期节点。
- **时间管理**：把计划放进日历，再用提醒和番茄钟推进执行。

## 基本使用

启用插件后，点击思源顶部栏的时间块日历图标打开日历。

默认写入路径形如：

```text
/daily note/YYYY/MM/YYYY-MM-DD
```

你可以在设置中修改笔记本 ID、日记根路径、通知、番茄钟和 AI 配置。

## AI 快速创建

顶部点击 **AI 助手**，输入自然语言，例如：

```text
明天上午 9 点到 11 点背单词，分类英语
6.8-6.30 每天各创建一个❌的全天事件
6 月 13 日六级考试，帮我创建倒数日，分类考试
从 2 月 28 日开始考研复习，创建正向日
10 月 30 日是我的生日，创建纪念日
6 月 11 日微机电考试，6 月 13 日六级，6 月 26 日嵌入式期末考试
```

AI 会返回结构化 JSON。插件会先校验并显示预览，只有你点击确认后才会写入时间块或倒数日。

## OpenAI-compatible 配置说明

在插件设置中启用 AI，并填写这些参数：

| 参数 | 作用 | 推荐填写 |
|---|---|---|
| API Base URL | 服务商的接口根地址。不要带 `/chat/completions`。 | DeepSeek：`https://api.deepseek.com/v1` |
| API Key | 服务商密钥，仅保存在本地插件设置中。 | 你的 API Key |
| 模型名称 | 用来解析日程的模型名。 | `deepseek-chat` |
| Temperature | 控制随机性。日程解析需要稳定，建议低一点。 | `0.1` |
| Max Tokens | 限制 AI 最多返回多少内容。批量识别很多安排时可调高。 | `2000` |
| 请求超时 ms | 等待 AI 响应的最长时间，单位毫秒。网络慢可调大。 | `30000` |

支持 DeepSeek、硅基流动、OpenRouter、本地转发服务等 OpenAI-compatible API。

## 数据与安全

- API Key 仅保存在插件本地设置中。
- 插件不会把 API Key 打印到控制台。
- AI 不会直接写入数据，必须经过预览确认。
- 删除时间块或倒数日时，插件会弹出确认提示。
- JSON 导入导出用于备份、迁移和恢复。
