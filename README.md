# 时间块日历

一个面向思源笔记的时间块日历插件，提供类似 Notion Calendar 的日程视图，用于在日记和普通文档中管理时间块、待办、提醒和番茄钟。

## 功能

- 周视图、三天视图、月视图、年视图。
- 拖拽创建时间块。
- 拖动时间块调整日期和开始时间。
- 拖动时间块顶部或底部调整开始、结束时间。
- 支持全天事件。
- 支持标签、颜色、完成状态和备注。
- 支持提醒和桌面系统通知。
- 支持 JSON 备份导入导出。
- 侧栏内置轻量番茄钟。
- 支持 iHour `.ihbak` 解析并导出为 JSON。

## 使用

安装并启用插件后，点击思源顶部栏的时间块日历图标即可打开日历。

插件会把时间块保存为思源文档中的记录表格，并同步写入块属性。刷新、重启或重新打开日历后，会从思源文档和块属性中重新读取时间块。

## 数据安全

插件通过思源内核 API 读写文档、块属性和插件配置，不直接操作思源工作空间的数据文件。

删除时间块会删除对应的日历记录；执行删除前，插件会显示确认提示。
